# Assets de audio locais

Quando os arquivos finais estiverem disponiveis, adicione-os em `app/src/main/res/raw/` com nomes em minusculo:

- `breath_in.ogg`
- `breath_out.ogg`
- `success.ogg`
- `transition.ogg`
- `grounding_tone.ogg`
- `rain_short.ogg`
- `ocean_short.ogg`
- `forest_short.ogg`
- `wind_light.ogg`

O aplicativo tambem possui audio procedural via `AudioTrack` para operar offline sem depender desses arquivos.
